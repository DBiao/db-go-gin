### 使用技术：
语言：Golang
数据库：MySQL

ORM框架：Gorm
配置框架：Viper
权限管理框架：Casbin
用户认证：Jwt
接口文档：Swagger


### 项目目录简介
项目结构遵循：https://github.com/golang-standards/project-layout

conf:         服务配置
pkg:          服务共有代码
docs:         swag文档
static:       保存的静态文件
internal:     服务私有代码
logs:         日志文件
script:       脚本文件
